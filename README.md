# Number Guessing Game
A classic, interactive, console-based number guessing game built using Python. The application generates a random number between 1 and 100 and guides the player to the correct answer with higher/lower hints.

# Features
Randomized Gameplay: Generates a new secret number every game.
Interactive Feedback: Provides real-time "smaller" or "larger" hints.
Input Validation: Prevents crashes from accidental text inputs.
Score Tracking: Counts the total number of attempts.

# Built With
Python 3 (using the built-in random and time modules)

# How to Run & Play
Run the script:
python main.py

Play: Type your guess into the terminal. Follow the higher/lower hints until you guess the correct number and see your total score.
